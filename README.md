# 仿 youtube 靜態頁面
<br />

> 作品介紹 
* 看了一些自學的文章，上面說進步的方式，是可以找一些主流的頁面模仿切版，所以找了 Youtube 當模板，模仿其介面樣式去練習切版，並嘗試以 Github Pages 以外的方法去部屬，最後選擇了 render.com
* 也嘗試不使用任何css框架，只使用原生css去切出類似youtube的樣式，此作品沒任何功能，只是一個靜態頁面去練習切版

<br />

> 使用技術
+ Nuxt 
+ Css 
+ RWD 
+ 部屬 render.com
